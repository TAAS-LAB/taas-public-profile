# GitHub Profile Setup Checklist

Use this checklist to finish the GitHub organization and repository presentation in the GitHub UI.

- [ ] Set the organization description to: `High-throughput transformation platform for Kafka Connect pipelines`
- [ ] Set the organization website to your primary domain or landing page
- [ ] Set the organization email to your public contact address
- [ ] Set the organization location
- [ ] Upload a clean avatar or logo
- [ ] Add the organization README from `org-profile/README.md`
- [ ] Pin the top 6 repositories in the order recommended in `docs/pinned-repos-plan.md`
- [ ] Add repository topics to the main repos, such as `kafka-connect`, `stream-processing`, `data-infrastructure`, `postgres`, `etl`, and `benchmarks`
- [ ] Make the key repositories public before sharing with reviewers
- [ ] Add a license to each public repo
- [ ] Add issue templates later once the public repo set is stable
